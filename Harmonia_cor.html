Salvar ou instalar como aplicativo

PC / Notebook

Verificando se o navegador permite instalação automática…

  1. Use Google Chrome ou Microsoft Edge.
  2. Quando o botão acima estiver disponível, clique em Instalar agora.
  3. Confirme a instalação.
  4. A ferramenta abrirá como aplicativo independente.

Se o botão não ficar disponível, use o ícone de instalação na barra de endereço ou o menu do navegador: ⋮ → Salvar e compartilhar → Instalar página como app.

Android

Verificando se o Chrome permite instalação automática…

  1. Abra esta página no Google Chrome.
  2. Quando o botão acima estiver disponível, toque em Instalar agora.
  3. Confirme a instalação.
  4. O app ficará disponível na tela inicial.

Se o botão não aparecer, toque no menu ⋮ e escolha Adicionar à tela inicial ou Instalar app.

iPhone / iPad

  1. Abra esta página no Safari.
  2. Toque no botão de compartilhar.
  3. Escolha Adicionar à Tela de Início.
  4. Confirme em Adicionar.

No iPhone, a Apple não permite que um botão da página instale automaticamente. A instalação precisa ser feita pelo menu de compartilhamento do Safari.
